四个维度提高自己

分手也有一段时间了，得走出去了。
发现自己身上很多的问题。
得感谢她，发现自己身上很多的问题。
也很对不起她，真的反反复复很多次的伤害了她。
先做好自己，提高自己。
过个几年再考虑追她了。
现在只爱她一个，但是自己问题确实太大了。

稳住，心态不能炸

知识水平和能力（经济）

不仅要多读书，也要多写作。
多输出观点和看法。
学会交流，完善自身的思想。
写博客，写文章。

专业知识水平

提升专业知识水平，为未来奠定基础。
首先是绩点、英语、项目、ACM。

多写代码
    提高自己的专业素养

综合知识水平

多读书
多接触一些时政历史地理情感金融方面的书
广泛的学习，有所涉猎

全面提高自己

提高情绪控制能力（心理）

提高情绪控制能力。
先去学会开导对方，倾听对方意见。
有观点不同的地方，不要先反驳，而是学会理解。
学会忍让，从心里上接受不公平，正确认知自己，调动积极性，拒绝消极沉沦

心态建设

坚持不懈，乐观向上
革命虽然难，但是终将成功

心态上，概括起来就是：乐观、自信、从容、不卑不亢。
